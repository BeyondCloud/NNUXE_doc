NNUXE
===================================

Neural Network  User eXperience Eumulator(NNUXE) is an flexible and modulized
framework for emulating user experience in AI edge deployment. NNUXE not only
provides an automatic pipeline for benchmarking and regression testing, the more
importantly, it will help Skymizer collect and gather users' needs and demand
and strengthen Skymizer core competitiveness.
